# 🥪 Бутербродная

Карточная игра на 2–4 игрока. Локальный мультиплеер — передавайте телефон по кругу.

## 🎮 Играть

Открыть в браузере (телефон или десктоп) — никакой установки.

## Правила

- 8 карт-персонажей: Батон, Брат батона, Яйко, Кетчуп, Маслён, Листок, Сырок, Поме
- На столе сначала лежит Батон. По кругу каждый играет одну карту.
- **Брат батона** → мгновенная победа
- **Яйко** → выбираешь противника, он пропускает ход, ты ходишь снова
- **Кетчуп** → можно класть только на Батон, Сырок или Поме. Иначе вылет.
- **Маслён** → можно класть только на Батон. Иначе вылет.
- Победа: первым сбросить все карты ИЛИ сыграть Брат батона.
